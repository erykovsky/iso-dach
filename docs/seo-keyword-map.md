# SEO Keyword Map (ISO-DACH)

## Założenia
- Priorytet: intencja transakcyjna (wycena/usługa) + lokalna (woj. zachodniopomorskie).
- GEO: treści odpowiadające na konkretne pytania użytkownika, z frazami usługowymi i lokalnymi.
- Główne klastry: izolacje, naprawy, termowizja, termomodernizacja, treści poradnikowe.

## GEO (AI search) - format odpowiedzi na stronach usług
- Pierwszy akapit: bezpośrednia odpowiedź „co to, dla kogo, jaki efekt”.
- Sekcja FAQ: 3-6 pytań problemowych z konkretną odpowiedzią.
- Konkretne dane i liczby (zakres czasu, grubości, etapy, orientacyjne oszczędności).
- Lokalne sygnały pod koniec sekcji CTA (obszar działania + kontakt).
